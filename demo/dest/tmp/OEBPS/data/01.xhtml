はじめに

このEPUB書籍は grunt-mdeb のデモを兼ねたドキュメントです。

grunt-mdeb とは

grunt-mdeb は EPUB 3.0 で電子書籍を制作する為の Grunt タスクです。 （名前は Markdown to eBook を略したものです）

主な機能

  • Markdown ドキュメントを元に EPUB ドキュメントを生成します。
  • ナビゲーションドキュメント（nav.xhtml / toc.ncx）はコンテンツドキュメントを元に自動生成されます。
  • その他のリソース（画像やスタイルシート）は便宜、指定のディレクトリに格納する事で同梱されます。

対応環境

タスクの実行には node.js と Grunt の環境が必要です。

  • node.js
    http://nodejs.org/
  • Grunt: The JavaScript Task Runner
    http://gruntjs.com/

また、書き出しにシェルスクリプトを使用する関係から、 Bash などが使える環境で Grunt タスクを実行する必要があります。 その為、Windowsのコマンドプロンプト等では使用出来ません。

  • Linux
  • MacOS X
  • Windows上のMingw（Git CUI） や cygwin 環境など
